## 1. 命令

```zsh
sudo apt list --installed # 查詢已安裝插件清單

sudo apt install # 安裝該插件
sudo apt remove # 解除安裝該插件

sudo apt update # 更新apt源

sudo apt upgrade # 升級所有插件

sudo apt autoclean # 清理暫存檔

sudo apt autoremove # 清理依賴
```
